Key Features

Zero-Copy RTCD Integration — Seamlessly integrated with AOM's runtime CPU dispatch. No modification to original encoder pipeline.
Automatic Device Selection — Intelligent scoring algorithm selects optimal GPU at runtime. Falls back to CPU gracefully.
Cross-Platform Support — Single codebase runs on NVIDIA (CUDA), Intel (Level Zero), AMD (ROCm), and ARM (OpenCL).
Production-Ready — SYCL 2020 compliant. Supports Intel DPC++ and AdaptiveCpp compilers. CMake build system.

Quick Start

Build & Run Bash
1# Clone repository
2git clone https://github.com/hbliu007/avm-sycl-gpu-acceleration
3cd avm-sycl-gpu-acceleration
4
5# Configure with CMake
6mkdir build && cd build
7cmake .. -DSYCL_PLATFORM=nvidia
8make -j$(nproc)
9
10# Run encoder
11./avmenc input.y4m -o output.av2 --cpu-used=4
